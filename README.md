# 🌸 Iris Flower Classification using Machine Learning
An end-to-end machine learning project using the Iris dataset for flower classification with data preprocessing, visualization, Random Forest, and Scikit-learn.

## 📌 Project Overview

This project demonstrates the complete machine learning workflow for solving a **classification problem** using the famous **Iris Dataset**.

The model is trained to classify iris flowers into three different species based on their physical measurements:

- 🌼 Setosa
- 🌺 Versicolor
- 🌸 Virginica

The project covers the entire machine learning pipeline from data loading to model evaluation and visualization.

# 🎯 Objectives

- Load and explore the Iris dataset
- Perform data preprocessing
- Check for missing values
- Visualize the dataset
- Split the dataset into training and testing sets
- Train a Machine Learning classification model
- Predict flower species
- Evaluate model performance
- Visualize the results

# 📂 Dataset

The project uses the built-in **Iris Dataset** provided by Scikit-Learn.

### Dataset Information

- Total Samples: **150**
- Features: **4**
- Classes: **3**

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Classes

- Setosa
- Versicolor
- Virginica
